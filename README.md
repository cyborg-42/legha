# ⚔️ LEGAHO — The Ember Wanderer

> *"To carry fire through darkness."*

A 2D Java game built from scratch as a hands-on journey to learn the Java programming language.  
Every phase of the game teaches a new set of Java concepts — no game engines, no shortcuts, just pure Java.

---

## 🎮 About the Game

In a world where light was stolen by the Void Lords, a lone wanderer carries the last ember —  
a fragment of the ancient sun sealed inside a rune. That wanderer is **you**.

LEGAHO is a 2D side-scrolling platformer built entirely in Java using `JFrame`, `JPanel`, and core Java libraries.  
No LibGDX. No Unity. Just Java the way it was meant to be learned — by building something real.

## 📚 What I'm Learning

### Phase 1 — Java Basics
- Variables, data types, conditionals, loops
- Classes and Objects
- Opening a window with `JFrame` + `JPanel`

### Phase 2 — OOP + Game Loop
- Inheritance, interfaces, abstract classes
- `ArrayList`, `Thread`, `Runnable`
- 60 FPS game loop (update + render cycle)

### Phase 3 — Input + Movement
- `KeyListener`, `KeyEvent`
- `HashSet` for multi-key input
- Player movement, gravity, jumping

### Phase 4 — Collision + Enemies
- `Rectangle.intersects()` for collision detection
- Enums for game states (`PLAYING`, `GAME_OVER`)
- Health system and Game Over screen

### Phase 5 — Sprites & Audio
- `BufferedImage`, `ImageIO` for sprites
- Sprite sheet animation
- `javax.sound.sampled` for music and SFX
- Exception handling (`try` / `catch`)

### Phase 6 — Levels & Shipping
- 2D arrays for tile maps
- File I/O — save and load high scores
- Packaging as a runnable `.jar`

## 🚀 How to Run

1. Clone the repo:
```bash
git clone https://github.com/YOURNAME/legaho-game.git
```
2. Open the project in **IntelliJ IDEA**
3. Set the JDK to **Java 21**
4. Run `Main.java`

> Once the game is packaged (Phase 6), you'll be able to run it with:
> ```bash
> java -jar legaho.jar
> ```
